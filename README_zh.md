# 密碼強度檢查程式

這個 Python 腳本旨在根據各種安全標準評估密碼的強度。它評估密碼的複雜性並提供如何改進的反饋。

## 功能

該腳本檢查以下標準來確定密碼強度：

- 最少8個字符長度。
- 至少一個大寫字母（`A-Z`）。
- 至少一個小寫字母（`a-z`）。
- 至少一個數字（`0-9`）。
- 至少一個特殊字符（`!@#$%^&*()_+{}[]:;<>,.?~\/-`）。
- 沒有連續或重複的字符。

## 使用方法

要使用該腳本，只需在 Python 環境中運行它。系統將提示您輸入密碼，之後腳本將評估其強度並提供反饋。

```bash
python password_strength_checker.py
```

## 需求

- Python 3.x

## 反饋提供

該腳本將密碼分為兩個強度等級：**高強度** 和 **低強度**。對於被認定為低強度的密碼，將提供具體原因以指導改進。
